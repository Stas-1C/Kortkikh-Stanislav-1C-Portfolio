Реализованный функционал
Интерфейс ввода: На управляемой форме размещена таблица значений с колонками: «Машина», «Дата С», «Дата По», «Адрес».
Добавление данных: Реализована кнопка «Добавить», которая позволяет вручную вводить произвольные данные в строки таблицы.
Визуализация: При нажатии кнопки «Прорисовать» обработка анализирует данные из таблицы и формирует график в поле табличного документа.
Логика отрисовки:
Каждая строка таблицы превращается в одну строку на графике.
Временной интервал (между «Дата С» и «Дата По») разбивается на ячейки по 30 минут.
Внутри нарисованного блока отображается введенный пользователем «Адрес».
