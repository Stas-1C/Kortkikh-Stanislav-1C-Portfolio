Это расширенная и более полная версия проекта из папки «Рукописная Своя». Данная конфигурация разработана специально для автоматизации всех аспектов деятельности одного торгового предприятия — ООО «Василёк».

Ключевые отличия и новые возможности
Полная структура: в отличие от базовой версии, здесь реализованы все основные объекты конфигурации, включая подсистемы, роли и права доступа.
Расширенный учёт: доработан механизм финансового учёта:
реализован механизм автоматического формирования бухгалтерских проводок при проведении торговых документов, что обеспечивает более тесную интеграцию управленческого и финансового учёта.
Единое предприятие: вся логика и структура данных сфокусированы на бизнес-процессах конкретной компании.
Что реализовано в этом проекте
Объекты конфигурации: справочники, документы, регистры сведений и накопления, отчёты, обработки.
Безопасность: настроена гибкая система ролей и прав доступа для разных групп пользователей (Администратор, Бухгалтер, Расчетчик, Мастер, Менеджер, Руководитель).
Автоматизация: реализованы сложные алгоритмы проведения документов и формирования движений по регистрам.
Проект демонстрирует способность создавать комплексные, готовые к внедрению решения «под ключ».
